# Dijkstra's algorithm

is an algorithm for finding the shortest paths between two nodes in a weighted graph, which may represent, for example, a road network. It was concieved by computer scientist Edsger W. Dijkstra in 1956.

Definition: Dijkstra's algorithm finds the shortest path from a given source node to every other node.[7]: 196–206  It can be used to find the shortest path to a specific destination node, by terminating the algorithm after determining the shortest path to that node

Data Structure: min-priority queue data structure
